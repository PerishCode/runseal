# v0.4.6-beta.1

- Add per-invocation log files under `ENVLOCK_LOG_HOME` or `ENVLOCK_HOME/logs`.
- Route Rust CLI, plugin host, and `node.sh` events into the same session log file.
- Improve `plugin node` observability around tool resolution, versions, state layout, lock events, and patch emission.
- Preserve plugin exit codes through the CLI so automation can distinguish lock contention and other plugin failures.
