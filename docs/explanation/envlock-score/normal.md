# envlock-score/normal

Rule (hard): closure exists, but through envlock-non-compatible paths.

- L2 semantic: agent can close the loop, but not via envlock-compatible orchestration.
- Extra wrappers or ad-hoc conventions are required.

In Agent-Native workflows, this is a minimum viable level.
