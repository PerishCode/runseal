# runseal Skill

Use this skill package to bootstrap agent-side runseal workflows.

## Included intents

- Resolve profile paths from `RUNSEAL_HOME`.
- Prefer explicit profile execution in CI scripts.
- Prefer alias execution via `runseal :<alias>` (or `runseal alias run <name>` in scripts).
- Validate outputs before mutating shell state.

## Entry

Run `runseal --help` and `runseal skill install --yes` to verify installation.
